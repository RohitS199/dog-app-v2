Pattern C · Hybrid

Both entry points open the trophy, always land on grid

Combines A's quick-access with B's canonical grid. Trophy is reachable from both surfaces; dismissing always lands on the grid as a discovery surface.

1
Profile root. Tap top-3 sticker → trophy detail opens directly (matched-position from profile row).
2
Trophy detail. Tap to dismiss → opens grid (not the profile root) — grid is the orientation surface.
3
Grid view. Tap any sticker → trophy detail (matched-position from grid cell). Dismiss returns to grid.
4
Close grid (X). Returns to profile.

Tradeoffs: Quick path from top-3 (one tap to trophy), but the grid becomes the always-on dismiss target. May surprise users who expected "back to profile" after dismiss from a row-launched trophy. Two origin sources to wire (row + grid). Always-to-grid dismiss makes the grid feel like a discovery hub.
